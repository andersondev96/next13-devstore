# Design System — Lumen

## Visão geral

O novo Design System da Lumen foi criado para transformar a interface em uma experiência mais premium, moderna e consistente, mantendo a simplicidade do projeto original.

## Princípios

- Clareza visual: interfaces objetivas e fáceis de navegar.
- Consistência: espaçamentos, cores, tipografia e componentes reutilizáveis.
- Escalabilidade: o sistema funciona para páginas de catálogo, produto e busca.
- Acessibilidade: contraste adequado, foco visível e componentes com estados claros.

## Tokens de identidade

### Cores

- Fundo principal: slate-950
- Superfície: slate-900/80
- Texto principal: slate-50
- Texto secundário: slate-300
- Marca: sky-400 / cyan-300
- Destaque de ação: brand-500

### Tipografia

- Família: Inter, com suporte nativo via Next.js.
- Títulos: semibold e com alto contraste.
- Texto de apoio: tamanho menor e linhas mais folgadas.

### Espaçamento e raio

- Raio de cards: 28px
- Botões: 9999px
- Sombras: suaves e discretas para dar profundidade.

## Componentes base

- Button
- Card
- SectionTitle

## Diretrizes de uso

- Use botões primários para ação principal.
- Use cards para organizar conteúdo e produtos.
- Mantenha consistência nas páginas de listagem e detalhe.

## Implementação atual

A base visual foi aplicada ao layout principal, ao header, às páginas de home, produto e busca, além de componentes de ação como botão de adicionar ao carrinho.
